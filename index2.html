Berikut adalah langkah-langkah untuk memulai:

  • Buka Android Studio
  • Pada gambar yang tertera di atas, klik Get from VCS di dekat bagian pojok kanan atas
  • Kemudian tampilannya akan menjadi seperti di bawah ini:
  • Pada bagian URL, isikan dengan URL yang didapatkan dari halaman ini, dengan cara:
  • Click Copy Link pada salah satu tombol yang tersedia dibawah
  • Buka Android Studio, lalu paste link tersebut ke bagian URL tersebut dan klik Clone
  • Kemudian tampilnya akan menjadi seperti di bawah ini:
  • Tunggu hingga proses Gradle Building... selesai, dapat dilihat di pojok kanan bawah
  • Proses selesai, jika anda ingin mengedit Tampilan(layout), silahkan pergi ke App -> res -> layout -> activity_main.xml
  • Jika anda ingin mengedit Warna (pada Button), silahkan pergi ke App -> res -> values -> colors.xml
